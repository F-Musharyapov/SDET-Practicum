ID: 1
Заголовок: Успешное создание клиента
Приоритет: High (Высокий)


Шаги:
1. Перейти на страницу https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager.
2. Нажать кнопку Add Customer.
3. Заполнить поле Post Code, сгенерировав номер из 10 цифр.
4. Заполнить поле First Name, сгенерировав имя на основе введенного Post Code: 
* Post Code условно разбить на двузначные цифры (получится 5 цифр)
* Каждую цифру преобразовать в букву английского алфавита по порядку от 0 до 25.
5. Заполнить поле Last Name валидным значением.
6. Нажать кнопку Add Customer.


Ожидаемый результат: Выводится всплывающее окно с сообщением “Customer added successfully with customer id :{id}” с кнопкой ОК.




ID: 2
Заголовок: Сортировка клиентов по убыванию First Name
Приоритет: Middle (Средний)
Предусловие: Таблица Customers содержит несколько строк с различными значениями поля First Name


Шаги:
1. Перейти на страницу https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager.
2. Нажать кнопку Customers.
3. В табличной части нажать на заголовок первого поля “First Name”.


Ожидаемый результат: Данные в табличной части сортируются по убыванию данных в поле First Name.

ID: 4
Заголовок: Успешное удаление клиента
Приоритет: High (Высокий)
Предусловие: Таблица Customers содержит несколько строк с различными значениями поля First Name


Шаги:
1. Перейти на страницу https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager.
2. Нажать кнопку Customers.
3. Нажать кнопку Delete для клиента с именем, длина которого ближе к среднему арифметическому длин всех First Name клиентов.


Ожидаемый результат: Успешное удаление из табличной части соответствующей строки с данными клиента.
